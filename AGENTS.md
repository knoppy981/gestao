## Key locations

- `lib/prisma.ts` — Prisma client singleton (Supabase PostgreSQL, via `@prisma/adapter-pg`). Import it with `@/lib/prisma`; never instantiate `PrismaClient` directly.
- `prisma/schema.prisma` — data models. In Prisma 7 the connection URLs are **not** here.
- `prisma.config.ts` — Prisma CLI config; holds the datasource URL (`DIRECT_URL`, session pooler) used for migrations.
- `.env` (gitignored) — `DATABASE_URL` (transaction pooler `:6543`, runtime) and `DIRECT_URL` (session pooler `:5432`, migrations).
- `generated/` — generated Prisma client output. Gitignored; regenerated by `prisma generate`. Do not edit or import it directly.
- `app/` — Next.js App Router routes, layouts, and pages.
